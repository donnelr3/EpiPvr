# EpiPvr

**Tools for modelling plant virus epidemiology**
  
  EpiPvr provides functions and workflows for plant pathogen profiling and epidemiological analysis with limited data. This package was developed to support parameter estimation and modelling of cassava virus systems, but can be applied more widely.
